# Comment Templates
Please use these templates when posting to a PR. This standard structure will enable us to perform analytics on community engagement and performance more easily.

## Started Working on a PR

```text
Started working on this PR.
```

## Stopped Working on a PR

```text
Stopped working on this PR.

| | |
| :-- | :-- |
| Reason | why work stopped |
| Current status | what is done / not done>|
```

## Validation Review Summary

```text
Validation summary:

| | |
| :-- | :-- |
| Decision | Accept / Modify |
| Confidence | Low / Medium / High |
| Summary | 1-2 sentances |
| Next step | merge / revise / re-review |
```

## Rejection Review Summary

```text
Rejection summary:

| | |
| :-- | :-- |
| Decision | Reject |
| Reason | 1-2 sentances |
| Blocking issues | short list |
| To reconsider | what must change |
```

## Duplicate Report

```text
Duplicate report:

| | |
| :-- | :-- |
| Decision | Duplicate |
| Parent PR | #123 (or https://github.com/org/repo/pull/123) |
| Notes | brief explanation of why this is a duplicate |
```
